# Main experiments

This directory contains the reproducibility scripts for the main rectangular
matrix reordering benchmarks in the VIS short paper.

Current contents:

- `synthetic_benchmark/`: procedural synthetic benchmark with representative
  rectangular reordering baselines.
- `real_world_benchmark/`: real-world rectangular benchmark with raw public
  data snapshots, preprocessing code, paper-table outputs, and qualitative
  figures.

The generated paper-table artifacts are stored under:

- `output/main_experiment/synthetic_benchmark/`
- `output/main_experiment/real_world_benchmark/`
