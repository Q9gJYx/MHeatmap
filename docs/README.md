# docs

Documentation and collaboration notes for VIS 2026 short paper preparation.

1. Main experiments (Sec 5; Table 1) (`main_experiment`)

- `main_experiment/synthetic_benchmark` contains the synthetic benchmark code.
- The generated synthetic paper-table artifacts are stored under
  `output/main_experiment/synthetic_benchmark`.
- `main_experiment/real_world_benchmark` contains the real-world benchmark
  preprocessing, evaluation, and figure-generation code.
- The generated real-world paper-table artifacts, cleaned matrices, and figures
  are stored under `output/main_experiment/real_world_benchmark`.

2. Tabula Sapiens Example (Sec 5.2; Fig 3) (`examples/tabula_sapiens`)

- This example can be replicated using `examples/tabula_sapiens/tabula_sapien_v2.py` script
- The datasets can be downloaded from: https://figshare.com/articles/dataset/Tabula_Sapien_v2/27921984
- Remember to change the dataset routes in the script to your own paths.
